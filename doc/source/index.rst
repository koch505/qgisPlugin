OpenGeo Explorer plugin for QGIS
================================

The OpenGeo Explorer plugin for QGIS is used to configure the components of OpenGeo Suite through QGIS. It allows you to easily configure your OpenGeo Suite instance, from preparing data and styling to publishing directly to GeoServer, all through the QGIS interface.

This manual describes how to use the OpenGeo Explorer plugin.
